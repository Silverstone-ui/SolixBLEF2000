F2000 (767 PowerHouse)
======================

.. note::
    Some 767 PowerHouse units use a different, unencrypted BLE protocol that this class does
    not implement — if ``connect()`` fails with a ``BleakCharacteristicNotFoundError``, see
    :doc:`f2000_hardware_variant`.
