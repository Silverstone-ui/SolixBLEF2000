F2000 (767 PowerHouse) - alternate hardware variant
=====================================================

.. note::
    See :doc:`f2000_hardware_variant` for the full protocol write-up, byte-level
    field map, and how to tell whether your unit needs this class instead of
    :class:`~SolixBLE.F2000`.

.. note::
    Control (AC output, DC/Car socket output, power saving mode, light bar
    mode) has been captured and confirmed working against real hardware.
    Display, timers, and AC charging power control have not been captured
    yet - see :doc:`f2000_hardware_variant` for details.
